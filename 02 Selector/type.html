Selector

In CSS, a selector is a pattern used to select the HTML elements you want to style. It is the part of the CSS rule that targets specific elements in the HTML document. There are various types of selectors in CSS, each serving a different purpose. Here are some common types of selectors:

  1. Type Selector : Html element
  2. Class Selector : Element with a specific class attribute
  3. ID Selector : Element with a specific ID attribute
  4. Attribute Selector : Element with a specific attribute
  5. Universal Selector : All elements
